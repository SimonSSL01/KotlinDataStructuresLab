### ¿De qué trata este proyecto?
En este repositorio comparto la solución a **30 retos de programación** desarrollados en Kotlin. El objetivo principal fue practicar cómo organizar y manipular datos de diferentes formas, pasando desde lo más básico hasta algoritmos un poco más detallados.

### Conceptos aplicados
Para resolver estos ejercicios, utilicé varias herramientas clave del lenguaje:

*   **Listas y Arreglos:** Para manejar datos en orden, como listas de tareas o historiales.
*   **Mapas (Maps):** Muy útiles para contar frecuencias (como en el carrito de compras) o para diccionarios de datos.
*   **Conjuntos (Sets):** Los usé cuando necesitaba asegurar que los datos no se repitieran, como en una lista de contactos únicos.
*   **Matrices:** Trabajé con arreglos bidimensionales para simular el orden de un almacén o mover posiciones en un mapa.
*   **Algoritmos Varios:** Implementé desde cálculos de promedios y filtros de precios, hasta lógica de números primos y balanceo de paréntesis.

###  Cómo probar los ejercicios
El código está estructurado en funciones independientes llamadas `reto1()`, `reto2()`, etc. Para verlos en acción:
1. Abre el proyecto en **IntelliJ IDEA**.
2. Ve al archivo `Main.kt`.
3. Llama a la función que quieras probar dentro del `fun main()`.
4. Dale al botón de **Run** y verás los resultados en la consola.
